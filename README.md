# Kotlin Simple Event Bus

Simple event bus by using `MutableSharedFlow`

`build.gradle.kts`
```kotlin
repositories {
    mavenCentral()
    maven {
        url = uri("https://jitpack.io")
    }
}

dependencies {
    implementation("com.github.t2PeNBiX99wcoxKv3A4g:kotlin-simple-event-bus:0.0.76")
}
```
